Security Rule Manager

The security rule manager allows you to import and export security rules from your Qlik Sense environment. They can be exported to a JSON file, then re-imported to any environment.

Rules with matching names or IDs on import will be overwritten, if more than 1 rule matches a name and/or ID, the rule will fail to import. Rules will also fail to import if they have previously been imported, then deleted, as the ID will already exist as a deleted item in the database.
